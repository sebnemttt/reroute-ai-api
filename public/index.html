ReRoute AI — IRROPS Skorlama API

Bu, ReRoute AI'nin öncelik skorlama motorunun gerçek, canlı backend'idir. Turkish Airlines Terminal Girişim Hızlandırma Programı 2026 başvurusu kapsamında geliştirilmiştir.

Endpoint 1: Tek Yolcu Skoru

POST /api/score
Content-Type: application/json

{
  "tier": "Elite Plus",
  "connMin": 38,
  "cabin": "Business",
  "special": false
}

Endpoint 2: Toplu Yolcu Skorlama

POST /api/batch-score
Content-Type: application/json

{
  "passengers": [
    { "name": "Elif Yıldırım", "tier": "Elite Plus", "connMin": 38, "cabin": "Business", "special": false },
    { "name": "Sarah Connors", "tier": "Classic", "connMin": 70, "cabin": "Economy Comfort", "special": true }
  ]
}
